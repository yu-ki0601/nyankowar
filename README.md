# ねこねこ大戦争プロトタイプ

「にゃんこ大戦争」のプレイ感を完全再現した、React + TypeScript 製の本格タワーディフェンス・プロトタイプです。

## 🌍 ステージ紹介
全4章の構成。ステージごとに色彩だけでなく、BGMもガラリと変化します。

## 🐱 味方ユニット (全8種)
にゃんこ、タンク、バトル、キモねこ、ウシねこ、ネコノトリ、フィッシュ、トカゲ。
それぞれの特徴（リーチ、速度、特殊攻撃）を活かした戦略が楽しめます。

## 👹 敵ユニット (全4種)
| ユニット名 | 特徴 |
| :--- | :--- |
| **わんこ** | 基本の敵。数で攻めてくる。 |
| **へび** | 高速移動。波打つ動きで接近する。 |
| **カバちゃん** | 高HPの壁役。大きな口で範囲攻撃。 |
| **くま** | 最凶のボス。強力な腕のなぎ払いで前線を破壊。 |

## ✨ 技術的特徴
- **ピュアコード描画:** 画像ファイルを一切使用せず、全てコードによる計算で描画。
- **マルチレイヤーRenderer:** タイトル画面とバトル画面をシームレスに切り替える描画エンジン。
- **数学的アニメーション:** サイン波やベジェ曲線を用いた、滑らかで不気味なキャラクターの動き。

## 🎮 遊び方
1. **タイトル:** 黄金のボタンをクリックして開始。
2. **ステージ選択:** 章を選んで出撃。
3. **バトル:** 8種のユニットを駆使して敵の城を落とせ！
4. **戻る:** 画面右上のボタンから、いつでもステージ選択へ復帰可能。
