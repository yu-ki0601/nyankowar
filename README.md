# にゃんこ大戦争プロトタイプ

「にゃんこ大戦争」のプレイ感を React + TypeScript + Canvas で再現した完全動作プロトタイプです。

## 🌍 ステージ紹介
全4章からなるストーリー構成です。
1.  **第1章「昼の草原」:** 基本を学ぶ入門ステージ。
2.  **第2章「夕暮れの平原」:** 敵の数が増え始める中級ステージ。
3.  **第3章「深夜の森」:** 視界の悪い暗闇で猛攻を仕掛けてくる上級ステージ。
4.  **第4章「月面拠点」:** 究極の難易度を誇る最終決戦。

## ✨ 特徴
- **データ駆動型設計:** ステージやユニットのデータが完全に分離されており、拡張が容易。
- **フル・プロシージャル描画:** 画像ファイルを一切使わず、すべてコードでリアルタイム描画。
- **波形合成サウンド:** 音声ファイルも不使用。Web Audio API でレトロな 8bit サウンドを生成。

## 🎮 遊び方
1.  **ステージ選択:** 起動画面で挑戦したい章を選択。
2.  **ユニット召喚:** 画面下のボタンでお金を消費して出撃。
3.  **にゃんこ砲:** 100%溜まったら発射！オレンジ色のゲージ演出が目印。
4.  **働きネコ:** お金を溜めてレベルアップし、経済を回しましょう。

## 🛠 技術構成
- **Engine:** `requestAnimationFrame` + `useRef` による独自物理エンジン。
- **Audio:** `useGameAudio` カスタムフックによる一括サウンド管理。
- **Renderer:** パーツごとに分割された高度な Canvas 描画システム。
