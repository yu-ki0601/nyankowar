# にゃんこ大戦争プロトタイプ

「にゃんこ大戦争」のプレイ感を React + TypeScript + Canvas で再現した完全動作プロトタイプです。

## ✨ 特徴
- **フル・プロシージャル描画:** 画像ファイルを一切使わず、猫や背景、エフェクトをすべてコードでリアルタイム描画。
- **波形合成サウンド:** 音声ファイルも不使用。Web Audio API で 8bit 風のレトロなサウンドを生成。
- **高精度ゲームエンジン:** フレームレートに依存しない滑らかな動きと、バグのない安定した状態管理。

## 🎮 遊び方
### 1. 起動
画面中央の **「GAME START」** を押すと、BGMが鳴り、にゃんことわんこの戦いが始まります。

### 2. ユニット召喚
画面下のボタンをクリックしてお金を消費し、にゃんこたちを出撃させます。
- **にゃんこ ($50):** 素早い生産が可能な基本ユニット。
- **タンク ($150):** 圧倒的な体力を誇る、縦長で頑丈な壁ユニット。
- **バトル ($200):** 手に斧を持ち、高い攻撃力で敵を粉砕するアタッカー。

### 3. 戦略要素
- **働きネコUP:** お金に余裕がある時にレベルを上げると、収入スピードが上がります。
- **生産ゲージ:** 召喚後の青いメーターが100%になると、完了音（チャリン）と共に再生産が可能になります。
- **にゃんこ砲:** ゲージが満タンになったら発射！ビームで敵をなぎ払い、後方へノックバックさせます。

### 4. 勝敗
- 敵城のHPを0にすれば **VICTORY (勝利)**。ファンファーレが鳴ります。
- 自城のHPが0になると **DEFEAT (敗北)**。

## 🛠 技術構成
- **Engine:** `requestAnimationFrame` + `useRef` による独自物理エンジン。
- **Sound:** `AudioContext` によるサイン波、スクエア波、ノイズの合成。
- **Visual:** `CanvasRenderingContext2D` によるベクター描画。
