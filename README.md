# ねこねこ大戦争プロトタイプ

「にゃんこ大戦争」のプレイ感を再現した、React + TypeScript 製の完全動作プロトタイプです。

## 🌍 ステージ紹介
全4章からなるストーリー構成です。
1.  **第1章「昼の草原」:** 基本を学ぶ入門ステージ。
2.  **第2章「夕暮れの平原」:** 敵の数が増え始める中級ステージ。
3.  **第3章「深夜の森」:** 視界の悪い暗闇での上級ステージ。
4.  **第4章「月面拠点」:** 究極の難易度を誇る宇宙決戦。

## 🐱 ユニットラインナップ (全8種)
| ユニット名 | 特徴 |
| :--- | :--- |
| **にゃんこ** | 基本戦力。安価で使いやすい。 |
| **タンク** | 高耐久。縦長の体で前線を死守する壁役。 |
| **バトル** | 攻撃重視。手に持った斧で敵を粉砕。 |
| **キモねこ** | 異常に長い脚を持つ。圧倒的な射程を誇る。 |
| **ウシねこ** | 高速移動。戦線への復帰が非常に早い。 |
| **トリ** | 空中ユニット。羽ばたきながら範囲攻撃。 |
| **フィッシュ** | 高HPかつ攻撃力も備えた魚型ユニット。 |
| **トカゲ** | 爬虫類フォルム。遠距離から強力な一撃。 |

## ✨ 技術的特徴
- **高度な分離設計:** ユニット・ステージ・音声・描画がそれぞれ独立した専門ファイルで管理されています。
- **フルコード描画:** 画像や音声ファイルを一切使わず、Canvas と Web Audio API のみで世界を構築。
- **高精度エンジン:** `useRef` を活用した 60fps 演算により、安定したゲーム体験を提供。
