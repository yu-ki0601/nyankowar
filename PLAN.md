# Web版「にゃんこ大戦争」風ゲーム 開発最終状況

## 1. 技術スタック
- **フレームワーク:** React (TypeScript)
- **ビルドツール:** Vite
- **設計パターン:** データ駆動型アーキテクチャ
- **描画エンジン:** 分離型 Canvas Renderer

## 2. 実装済み機能ハイライト

### ステージ & 拡張性 🌍
- [x] **ステージ選択画面:** 起動時に挑戦するステージを選択可能。
- [x] **スケーラブル設計:** ステージデータを独立ファイル (`stage1.ts`, `stage2.ts`) に分離。
- [x] **難易度変動:** ステージごとに敵のHP倍率や湧き速度を設定可能。
- [x] **環境変化:** ステージごとに空、山、地面の色彩を動的に変更。

### ビジュアル & サウンド 🎨🎵
- [x] **ユニット描画の専門化:** `drawCat`, `drawDog` などの専用描画ファイル。
- [x] **リッチ演出:** 流れる雲、旗のなびく城、跳ねるキャラ、斧のスイング。
- [x] **サウンド合成:** BGM、攻撃音、召喚音、勝利/敗北ジングル、にゃんこ砲音。

### ゲームシステム ⚙️
- [x] **にゃんこ砲:** ゲージ連動型UIと範囲攻撃・ノックバック。
- [x] **働きネコ:** 経済アップグレードとコスト表示。
- [x] **生産ゲージ:** 本家風のクールタイム可視化。

## 3. 現在の構造
最新のリファクタリングにより、100ステージ以上の追加にも耐えうる、プロ仕様のメンテナブルなコードベースとなりました。
