# Web版「ねこねこ大戦争」風ゲーム 開発最終状況

## 1. 技術スタック
- **フレームワーク:** React (TypeScript)
- **描画エンジン:** HTML5 Canvas (パーツ別 Renderer & アニメーション)
- **音声エンジン:** Web Audio API (独自シンセサイザー & ステージ別BGM)
- **設計:** 責務分離型アーキテクチャ (UI / Logic / Audio / Visual)

## 2. 実装済み機能ハイライト

### ビジュアル & 演出 🎨
- [x] **リッチなタイトル画面:** 
    - 「ねこねこ大戦争」ロゴと、賑やかに跳ね回るネコたちの行進。
    - ゴールドに輝くタイトなデザインの「CLICK TO START」ボタン。
- [x] **高度なキャラクター演出:** 
    - **キモねこ:** 攻撃時に腕がムチのようにビュンと伸びる「超リーチ攻撃」を実装。
    - **トカゲ:** 低い姿勢、しなる長い尻尾、背中のトゲなど爬虫類らしさを徹底追及。
    - **共通:** 全キャラに「足元の影」と「瞳の光」を追加し、接地感と生命感を向上。

### 画面遷移 & システム ⚙️
- [x] **完全なゲームフロー:** 
    - タイトル ➔ ステージ選択 ➔ バトル ➔ リザルト ➔ ステージ選択 の循環を確立。
- [x] **8ユニット編成:** 2行×4列の機能的な操作パネル。
- [x] **ステージ別BGM:** 4つの章それぞれで、音色やリズムが全く異なる専用BGMを自動切り替え。

## 3. 現在の構造
エンジニアリング的にも視覚的にも「プロトタイプ」を超えた完成度に到達。
特筆すべきは、Canvas内でのアニメーション管理と React の状態遷移が見事に同期しており、非常にスムーズなプレイ体験を実現している点です。
