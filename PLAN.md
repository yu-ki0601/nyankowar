# Web版「ねこねこ大戦争」風ゲーム 開発最終状況

## 1. 技術スタック
- **フレームワーク:** React (TypeScript)
- **設計パターン:** 
    - **関心の分離:** 描画(Renderer)、音声(Synth/Hook)、定数(Constants)を完全分離
    - **データ駆動:** ステージ・ユニット情報を独立ファイルで管理
    - **安定エンジン:** useRef (stateRef) による 60fps 高速演算
- **描画:** HTML5 Canvas API (パーツ別 Renderer)
- **音声:** Web Audio API (再帰的スケジューラ & ステージ別 BGM)

## 2. 実装済み機能ハイライト

### ビジュアル & ステージ 🌍
- [x] **全4章のバリエーション:** 草原(昼)、平原(夕)、森(夜)、月面(宇宙)。
- [x] **リッチな背景演出:** 動きのある雲、遠近感のある山々、各ステージ専用の色彩。
- [x] **キャラクター造形:** 
    - 味方: 基本ねこ、縦長タンク、斧装備バトル。
    - 敵: ダークグレーの「黒わんこ」(赤目・垂れ耳)。
- [x] **ステージ選択:** 章ごとの難易度と雰囲気を楽しめる選択UI。

### サウンド & 演出 🎵
- [x] **ステージ別BGM:** 全4章で全く異なるメロディ・リズム・質感の音楽。
- [x] **高安定再生:** `setTimeout` 再帰呼び出しによるブラウザ制限に強い再生系。
- [x] **フルSE:** 打撃音(ガシッ)、召喚音、完了音(チャリン)、にゃんこ砲爆発音、勝敗ファンファーレ。

### システム ⚙️
- [x] **にゃんこ砲:** ボタン自体がオレンジ色に充填されるゲージ演出UI。
- [x] **働きネコ:** お金の増加速度を段階的に強化。
- [x] **本家風戦闘:** ユニットのスタック（重なり）と、正確な範囲ダメージ計算。
