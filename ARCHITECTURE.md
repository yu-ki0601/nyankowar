# ねこねこ大戦争: アーキテクチャ設計書

## 1. 画面状態管理 (GameState)
`gameState` により、Canvas の描画内容と UI オーバーレイを完全に同期させています。
- **title:** タイトルロゴ表示。Canvas内では背景と賑やかしキャラクターを自動描画。
- **start:** ステージ選択画面。
- **playing:** メインゲーム。物理演算エンジンが稼働。
- **victory / defeat:** リザルト表示。

## 2. 描画パイプラインの深化 (`renderer.ts`)
- **動的レイヤー:** `gameState === 'title'` 時は専用の背景描画関数 `drawTitleBackground` へ分岐。
- **高精度アニメーション:** 
    - キャラクターの描画関数（例：`drawCatLegs`）に `timestamp` を渡し、サイン波等を用いた複雑なパーツ変形（伸縮する腕、しなる尻尾）を実現。
    - React のステート更新に依存しない、Canvas 直書きによる 60fps の滑らかな動き。

## 3. 音響イベントシステム
- **Effect連携:** `Game.tsx` の `useEffect` が `gameState` の変化を監視し、曲の切り替えや停止を自動制御。
- **ディレクトリ構成:** 
    - `src/audio/music/bgm/`: 各ステージ固有の演奏ロジック（波形・旋律）。
    - `src/audio/music/bgm.ts`: 外部から ID を指定するだけの抽象化されたインターフェース。

## 4. UI設計
- **複合レイアウト:** 
    - 特殊コマンド（左側・縦並び）と、ユニット召喚（右側・グリッド）を分離。
    - `GaugeButton` による「再生産クールタイム」の視覚的フィードバック。
