# ねこねこ大戦争: アーキテクチャ設計書

## 1. キャラクター描画エンジン (`src/components/Canvas/units/`)
各キャラクターは独立したファイルで定義され、専門の描画ロジックを持ちます。
- **味方ユニット:** `CatBasic`, `CatLegs` (腕の伸縮), `CatLizard` (火炎放射) 等。
- **敵ユニット:** `DogEnemy`, `EnemyHippo` (噛みつき), `EnemySnake` (蛇行), `EnemyBear` (なぎ払い)。
- **アニメーション:** `timestamp` を利用した数学的補間（Sine波等）により、スムーズな非同期アニメーションを実現。

## 2. 背景・タイトルシステム (`renderer.ts`)
- **シーン分岐:** `gameState` に応じて「タイトル専用シーン」と「バトルシーン」を切り替え。
- **共有アセット:** タイトル画面でもバトル用の描画関数を再利用し、一貫性のあるビジュアルを提供。

## 3. 音響ライフサイクル
- **Effect同期:** React の `useEffect` で `gameState` と `currentStage.id` を監視。これにより、ステージ切り替えやメニュー戻り時のBGM停止・開始を漏れなく制御。
- **独立BGM:** ステージごとに波形（Sine, Square, Triangle）とリズムを完全に分けた独立演奏ロジック。

## 4. データ構造
- **マスターデータ:** `src/constants/units.ts` に全12種（味方8+敵4）のステータスを集約。
- **スポーンロジック:** `Game.tsx` 内で重み付け確率（わんこ50%, へび30%等）による動的エンカウントを実装。
